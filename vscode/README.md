# Fez VS Code Tools

Starter VS Code extension for Fez tooling.

## Run Locally

1. Open the `vscode-plugin` folder in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. Run `Fez VS Code Tools: Hello World` from the command palette.

## Commands

- `Fez VS Code Tools: Hello World` shows a confirmation message.
- `Fez VS Code Tools: Insert Timestamp` inserts an ISO timestamp into the active editor.
