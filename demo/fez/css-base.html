• Generates utility CSS classes for spacing, positioning, and sizing
  • Similar to Tailwind CSS utility classes, just add tag, you have mini tailwind
  • Classes without 'px' suffix use 4px increments (e.g., m-5 = 20px)
  • Classes with 'px' suffix use exact pixel values (e.g., m-5px = 5px)
  • Supports values from 0 to 50
  • Mobile/desktop specific classes available with prefixes
