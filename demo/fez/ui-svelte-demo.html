• Svelte-like Template Syntax - Uses single braces {expression} instead of double braces {{expression}}
  • Supports: {#if}, {:else if}, {:else}, {/if}
  • Supports: {#unless}, {/unless}
  • Supports: {#each items as item}, {/each} - implicit i available
  • Supports: {#each items as item, index} - explicit index name
  • Supports: {#for item in items}, {/for} - implicit i available
  • Supports: {#for item, index in items} - explicit index name
  • Object iteration (3 params): {#each obj as key, value, index}
  • Object iteration (3 params): {#for key, value, index in obj}
  • Object iteration (destructured): {#for [key, value] in obj}
  • Supports: {@html rawContent} for unescaped HTML
  • Supports: {@json obj} for debug output
  • Supports: onclick={()=>this.method()} event handlers
  • Event handlers in loops capture loop variables as closures (direct reference, not serialized)
  • Automatic brace-depth counting for complex expressions like {items.map(x => ({...x}))}
