• File extension icon component with ext parameter
  • Accepts extension, full filename, or URL - automatically extracts extension
  • Shows Phosphor duotone SVG icons (36x36) for common file types
  • Default colors by file type: images (blue), videos (purple), PDF (red), docs (blue), Excel (green), etc.
  • Default titles for each file type (e.g., "PDF Document", "JPEG Image", "Excel Spreadsheet")
  • Override defaults with color and title props
  • Falls back to text-based icon with uppercase extension for unknown types
