• Uses {#await} block to handle async data fetching
  • Shows loading spinner while promise is pending
  • Displays user card when promise resolves
  • Shows error message with retry button if promise rejects
  • 30% chance of simulated error to demonstrate error handling

Async User Profile

Syntax

{#await state.userData}
  <div class="loading">Loading...</div>
{:then user}
  <div>{user.name}</div>
{:catch error}
  <div>Error: {error.message}</div>
{/await}
